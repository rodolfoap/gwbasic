# GWBasic in a docker container

https://hub.docker.com/r/rodolfoap/gwbasic

## Usage

Get the container, tweak the `c/dosbox.conf` file according to your taste. Then run dosbox:

```
docker pull rodolfoap/gwbasic
./launch.bash
```
